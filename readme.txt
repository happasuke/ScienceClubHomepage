編集する人へ

・いままでの活動を書くとき
　index.htmlに直接書き足してください。

・RecentEvent（最近の記事）を書くとき
　./eventにテンプレがあります、利用してください。
最新のいくつかをトップページに表示してください。

・部内ロボコンのページの更新をするとき
　レイアウトを参考に、それまでの記事の最上部に追加してください。
　動画やサムネ、写真は　/ClubRobocon/image と movie　に追加してください。
  動画はなるべくyoutubeのリンクを張ってください

・ページ内リンクは適宜追加してください。


また、各ページのデザイン等のおおきな変更をするときは対象のページのバックアップをとったうえでいつ、誰が、どこを変更したのかを変更した後のページに明記することをおすすめします。
